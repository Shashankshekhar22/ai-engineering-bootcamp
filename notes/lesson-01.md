1.  Artificial Intelligence

    The broad field of making machines perform tasks that normally require human intelligence.

    Examples:
    Face recognition
    Voice assistants
    Fraud detection
    Chess engines
    Self-driving systems

2.  Machine Learning (ML)

    Instead of writing rules manually, the computer learns patterns from data.

    Example:
    Rather than hard-coding what spam looks like, you train a model on thousands of spam and non-spam emails.

3.  Deep Learning

    A specialized area of ML using neural networks with many layers.

    Deep Learning powers:
    _ Image recognition
    _ Speech recognition
    _ Translation
    _ LLMs

4.  Generative AI

    Instead of predicting a label, it generates new content.

    Examples:
    _ Text
    _ Images
    _ Code
    _ Audio
    _ Video
    _ ChatGPT is a Generative AI application.

5.  Large Language Models (LLMs)

    An LLM is a neural network trained on massive amounts of text to predict the next token (roughly, the next piece of text).

    Despite that simple objective, it can:

        * Answer questions
        * Write code
        * Summarize documents
        * Explain concepts
        * Generate structured data

6.  How is an AI application different from a traditional CRUD application?

An AI application does not works on a set of predefined rules. It does the thinking and analysis on the fly and take decision based on the data on which it is trained. Where as CRUD application has some set of predefined rules, it operated based on a database but an AI appplication does not work in the same way.

7. Why does an AI application still need a backend if the frontend can call an LLM directly?

An AI application still needs a backend because we want a processed data to the FE, along with that it adds some layer of Security which FE can't provide directly.

8. What new challenges do AI applications introduce (for example, response variability, latency, cost, reliability)?
