# Module 05 – Function Calling

## Module Objective

Learn how Large Language Models interact with external systems using Function Calling and build a production-ready tool execution workflow using the OpenAI SDK.

---

## Prerequisites

- Module 01 – AI Fundamentals
- Module 02 – LLM Fundamentals
- Module 03 – OpenAI SDK
- Module 04 – Structured Outputs

---

## Learning Outcomes

By the end of this module, I will be able to:

- Explain why Function Calling exists.
- Differentiate Function Calling from Structured Outputs.
- Design effective tool schemas.
- Implement Function Calling using the OpenAI Responses API.
- Build a reusable Tool Registry.
- Design an AI Orchestrator.
- Handle multiple tool calls.
- Implement production-grade error handling.
- Apply Clean Architecture principles.
- Integrate Function Calling into Career Copilot.
